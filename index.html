05 Sobre el Proyecto

📋 Contexto

Proyecto personal desarrollado desde cero para aprender sobre seguridad de redes, programación en Python y construcción de agentes de IA. Incluye validación de entradas, manejo de errores robusto y tests automatizados.

Compatible con Windows, Linux y macOS. El detector Deauth funciona en Linux con tarjeta WiFi en modo monitor.

🛠️ Stack Tecnológico

Python 3.8+ NMAP Scapy SQLite Requests unittest macvendors API Twilio
