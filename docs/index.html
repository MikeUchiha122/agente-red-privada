05 Sobre el proyecto
Contexto

Proyecto personal desarrollado desde cero para aprender sobre seguridad de redes, programación en Python y construcción de agentes de IA. Compatible con Windows, Linux y macOS.

Stack tecnológico
Python 3.8+ NMAP Scapy SQLite Requests unittest macvendors API Twilio
Seguridad del código

Validación estricta de IPs y rangos CIDR. Sanitización de inputs contra comandos maliciosos. Límites de concurrencia para evitar sobrecarga. Las API keys nunca se exponen en logs.
